predict - encapsulate position and velocity predictions

* posat(T,O) -- predict Body-to-Target vector a time T.

The parameters are the Universal Time,
and the Orbitable of interest. The return
value is a position, relative to the body
that the SHIP is currently orbiting.

The O parameter defaults to SHIP.

#+begin_src ks
  set posat to { parameter t. parameter o is ship.
  return positionat(o,t)-body:position.}.
#+end_src

* velat(T,O) -- predict Target Velocity relative to Body at time T.

The parameters are the Universal Time,
and the Orbitable of interest. The return
value is a velocity, relative to the body
that the SHIP is currently orbiting.

The O parameter defaults to SHIP.

#+begin_src ks
  set velat to { parameter t. parameter o is ship.
  return velocityat(o,t):orbit.}.
#+end_src
